# Governance Risk & Compliance Risk Assessment

This project demonstrates a Governance, Risk, and Compliance (GRC) assessment for a cloud-supported social services environment handling sensitive PII/SPII.

## Project Deliverables
- Executive Risk Summary
- Risk Register with Threat Sources and Asset Classification
- Risk Scoring and Mitigation Planning

## Framework Alignment
- NIST Risk Management Framework (RMF)
- Identity and Access Risk Analysis
- Vendor and Endpoint Risk Considerations
